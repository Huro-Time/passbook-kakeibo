# 通帳家計簿

スマホ上で動作する, 通帳風のシンプルな家計簿Webアプリです.

## 主な機能

- 通帳を月ごとに分けて表示します.
- 収入, 支出, 現在高を通帳欄に表示します.
- 月別の初期残高を設定できます.
- 初期残高を設定した月では, 通帳の先頭行に初期残高と現在高が記録されます.
- 金額入力はATM風のアプリ内テンキーで行います.
- CSV出力は削除済みです.
- 参考画像に近づけるため, 5列構成の通帳表, 赤系の罫線, クリーム色の用紙, 文字配置を調整しています.
- スマホ幅でも5列が収まるようにし, 通帳欄にはドットフォント風の表示を適用しています.
- バックアップと復元はJSON形式で行えます.

## 使い方

1. `index.html` をブラウザで開く.
2. 設定タブで月別初期残高と用途を設定する.
3. 記入タブで収入または支出を登録する.
4. 金額は画面内のATM風テンキーで入力する.
5. 通帳タブで月ごとの通帳を確認する.
6. 明細行をタップすると, 編集と削除ができます.

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成する.
2. このフォルダ内の `index.html` と `README.md` をアップロードする.
3. リポジトリの Settings から Pages を開く.
4. Source を Deploy from a branch にする.
5. Branch を `main`, Folder を `/(root)` にして保存する.
6. 数十秒から数分後に表示されるURLへアクセスする.

## 注意

入力データはブラウザの localStorage に保存されます. GitHubには送信されません. 端末変更前は設定タブからバックアップしてください.
